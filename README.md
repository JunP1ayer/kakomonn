# 過去問hub

大学生のための過去問 × 情報共有プラットフォーム

## 機能

- 📚 過去問アーカイブ：学部・授業名・年度で検索
- 💬 情報交換：Discord風のスレッド機能
- 🎯 パーソナライズ：大学・学部に合った情報配信
- 🏫 10大学対応：東京大学、京都大学、早稲田大学、慶應大学、大阪大学、名古屋大学、東北大学、北海道大学、九州大学、一橋大学
